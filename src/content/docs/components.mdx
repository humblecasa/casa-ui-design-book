# Component sizing defaults

Stop inventing heights. Pick a ramp and reuse it.

## Control heights
- `control-sm`: 32
- `control-md`: 40 (default)
- `control-lg`: 48

## Icons
- `icon-sm`: 16
- `icon-md`: 20 (default)
- `icon-lg`: 24

## Radius
- `radius-sm`: 6
- `radius-md`: 10 (default)
- `radius-lg`: 14

## Rule
Every component chooses `sm/md/lg`. If you need a new size, add it as a token intentionally.
