# Typography tokens (modular scale + roles)

Pick one scale and stick to it. A solid UI default ratio is **1.25**.

## Font size tokens (example)
- `text-xs`: 12 (caption)
- `text-sm`: 14 (labels)
- `text-md`: 16 (body)
- `text-lg`: 20 (subheading)
- `text-xl`: 24 (H3)
- `text-2xl`: 32 (H2)
- `text-3xl`: 40 (H1/hero)

## Line-height tokens
- `lh-tight`: 1.2 (headings)
- `lh-normal`: 1.5 (body)
- `lh-loose`: 1.7 (long reading)

Choose **roles**, not numbers.
